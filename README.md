# Tho0r-Craft - Guide d'Utilisation

## Introduction
Tho0r-Craft est un outil avancé de génération de prompts pour la création d'images par IA. Il est conçu pour les utilisateurs desktop et offre une interface riche en fonctionnalités pour affiner chaque détail de vos créations.

## Fonctionnalités Principales

### 1. Génération de Prompts
- **Catégories et Attributs :** Choisissez parmi diverses catégories (Humanoïde, Véhicule, Architecture, etc.) et personnalisez les attributs (coiffure, vêtements, accessoires, etc.).
- **Choix Multiples :** Certains attributs, comme les accessoires, permettent de sélectionner plusieurs options simultanément.
- **Tags Personnalisés :** Ajoutez vos propres mots-clés pour enrichir le prompt final.
- **Surprise Me :** Générez une configuration aléatoire complète en un clic pour stimuler votre créativité.
- **Enhance (IA) :** Optimisez automatiquement votre prompt grâce à l'intelligence artificielle.

### 2. Gestion de la Galerie
- **Glisser-Déposer (Drag & Drop) :** Réorganisez manuellement l'ordre de vos images dans la galerie en les glissant-déposant.
- **Projets / Dossiers :** Assignez un projet à vos images pour mieux les organiser et filtrez la galerie par projet.
- **Favoris :** Marquez vos meilleures créations comme favorites (icône cœur) pour les retrouver facilement.
- **Lightbox Interactive :** Cliquez sur une image pour l'agrandir, voir ses détails (prompt, tags, projet) et accéder à des actions rapides (copier le prompt, utiliser comme référence, télécharger).
- **Import / Export :** Sauvegardez l'intégralité de votre galerie au format JSON et restaurez-la quand vous le souhaitez.

### 3. Outils Avancés
- **Img2Img (Image to Image) :** Utilisez une image existante comme référence pour générer de nouvelles variations.
- **Comparaison :** Sélectionnez deux images dans la galerie pour comparer visuellement leurs prompts et identifier les différences.
- **Variations :** Générez 3 variations de prompt complètement différentes à partir de votre configuration actuelle.
- **Upscale :** Améliorez la résolution de vos images générées.

### 4. Presets et Historique
- **Presets :** Sauvegardez votre configuration complète (catégorie, attributs, tags, etc.) pour la réutiliser plus tard.
- **Historique (Undo/Redo) :** Annulez ou rétablissez vos dernières modifications de configuration.

## Raccourcis Clavier
- `Shift + Enter` : Générer une image
- `Ctrl + Enter` : Copier le prompt
- `Ctrl + G` : Générer des variations
- `Ctrl + S` : Exporter la galerie
- `Ctrl + Z` : Annuler (Undo)
- `Ctrl + Y` ou `Ctrl + Shift + Z` : Rétablir (Redo)
- `Alt + 1-7` : Changer de catégorie

## Conseils d'Utilisation
- Utilisez le mode **Focus** (icône d'agrandissement) pour masquer les panneaux latéraux et vous concentrer sur l'image générée.
- N'hésitez pas à combiner les options "Surprise Me" et "Enhance" pour découvrir de nouvelles idées de prompts inattendues.
